Soil Moisture Sensor

This project features a smart plant moisture monitor designed to help you keep your household plants thriving. Housed in a custom-designed, 3D-printed enclosure, the device uses an ESP32-WROVER microcontroller to accurately sense and display soil moisture levels, taking the guesswork out of watering.

NOAA Weather Satellite Receiver

This project demonstrates the reception and recording of NOAA weather satellite signals using an RTL-SDR V3 and GNU Radio. Through GNU Radio, the RTL-SDR is configured to capture RF data, which then undergoes Digital Signal Processing (DSP) for real-time visualization of signal characteristics such as Signal-to-Noise Ratio (SNR). The processed satellite data is subsequently recorded for further analysis or decoding.

RTL-SDR GNU Radio NOAA RF

Hydrogen Line Radio Astronomy

This project demonstrates the ability to create a cheap and easy hydrogen line radio telescope. The hardware setup for this project consists of a 2.4 GHz parabolic Wi-Fi grid antenna for reception, RTL-SDR for Digital Signal Processing, and a Low Noise Amplifier (LNA). SDRSharp software + a IF Average (Fast Fourier Transform) plugin is utilized for visualization and recording.
